# Polaris 🌟

A brain-training puzzle game collection for iOS.

## 🎮 Games
- **Wordle** — guess the hidden word in 6 attempts
- **Shikaku** — divide the grid into rectangles using number clues
- **Connections** — find the hidden link between groups of words
- *More coming soon...*

## 🚧 Status
Currently in early development.

## 🛠️ Built With

- Xcode
